21. század színvonalán

Több év tapasztalatunk van Java, Oracle SQL rendszerek építésében, valamint a legmodernebb JavaScript / Node.js alapú és Open Source fejlesztésekben. Ezen tapasztalatok alapján tudásunkat folyamatosan fejlesztjük és elsajátítjuk az új technológiákat. Tapasztalattal rendelkezünk abban, hogy a régi alkalmazásokat, hogyan lehet modernizálni, milyen módon érhető el egy Javas monolitikus rendszer felbontása és újraírása Node.js alapokon.

Az általunk tervezett, vagy fejlesztett rendszerek akár több tízezer kérést is kiszolgálnak másodpercenként anélkül, hogy ez jelentős hardver igényt támasztana.
